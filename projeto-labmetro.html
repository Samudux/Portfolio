REAL PROJECT INTERNAL TOOLING REACT IN PRODUCTION

LABMETRO VPS

Calibration Certificate Generator

Context

LABMETRO VPS is an accredited calibration laboratory operating under ABNT NBR ISO/IEC 17025:2017 — the international standard for testing and calibration laboratories. As part of each job, the lab issues a formal Calibration Certificate: a technically precise document recording equipment data, measurement results, uncertainties, standard references and environmental conditions.

Before this tool existed, every certificate was produced manually — a technician would fill a template field by field, calculate error percentages by hand, format two separate PDF pages, and merge them into a single file. The process took 20 to 30 minutes per certificate and introduced consistent, hard-to-detect errors.

Problem

Three compounding problems were identified. First, the average indication error was being recorded in measurement units instead of percentage — a direct ABNT violation that went undetected because the calculation was done manually each time. Second, certificates required two separate PDFs (SI and imperial units) that had to be manually merged or sent as two files. Third, standardized ABNT text blocks were typed from scratch on every certificate, creating wording variations that constitute a compliance risk under audit.

Solution

A browser-based web application where a technician fills a structured form and generates a fully formatted, two-page calibration certificate as a single unified PDF — in under two minutes. Calculations are automated and correct per ABNT. Unit conversion is handled automatically. Standardized text blocks are inserted verbatim. No backend, no installation, no data loss between sessions.

Technical Decisions

No backend

All computation and PDF generation happens in the browser. This eliminates hosting complexity and network dependency — critical for a lab environment where a stable connection cannot always be assumed. Draft persistence via localStorage allows resuming across sessions without data loss.

Dynamic-height PDF generation

Certificate length varies by number of measurement points. Forcing a fixed A4 height caused content to be clipped. Each page is measured at runtime (element.offsetHeight) and PDF dimensions are set dynamically — no content is ever cut off, regardless of certificate length.

Two-page unified export

Page 1 (N.m) is captured as a jsPDF instance. Page 2 (lbf.ft) is captured as a canvas and added as a new page with exact dimensions. This avoids the page-break detection issues that arise when exporting both pages in a single pass.

Calculation correctness

The average indication error is the arithmetic mean of individual percentage errors — not the percentage of the mean absolute error. These are mathematically different. The formula was confirmed against the client's own Excel reference model before implementation.

Delivery Process

The project was delivered in two formal feedback rounds. Round 1 covered the core feature set: unified PDF, correct error percentage, dual-unit display and structured form fields. Round 2 addressed compliance refinements: automatic ABNT text block insertion, signatures consolidated to the final page, and the accreditation line on page 2.

Each round was reviewed against a real certificate issued under Cgcre audit and the laboratory's internal Excel calculation model, used as ground truth for formula validation.

Results

20–30 min → under 2 min

Certificate generation time reduced by over 90%. The tool is in active use for real client jobs.

Calculation errors eliminated

Error percentage is now computed automatically and correctly per ABNT — a violation that previously went undetected in every manual certificate.

Audit-ready wording

Standardized ABNT text blocks are inserted verbatim on every certificate — no manual copy-paste, no wording variation across documents.

Single-file output

Both pages exported as one unified PDF. No post-processing, no risk of sending two separate files.
